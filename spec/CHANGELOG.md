# Spec — Change Log

Consolidated change history for the documents in `spec/` (and `spec/channels/`).
**Newest first.**

- One row per document revision. To record a change: add a row at the top and
  bump that document's frontmatter `status: draft (vX.Y)`.
- Authoritative history is git (`git log --follow -- spec/<file>`); this file is
  the curated digest. The four release specs (core, channels-general, custodian,
  interop) previously carried a prose `Appendix E/D — Change Log`; that prose is
  preserved in git and summarized below.

> **"CSS"**, **"Conversation Session Stream"**, or **"Conversational Session
> Synthesis"**, it denotes the *same* Channel 7 now called **SCS**. Historical
> rows are preserved verbatim and are not edited retroactively.
